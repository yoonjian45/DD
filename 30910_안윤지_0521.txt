데이터 정의어(ddl):객체를 새로 만들거나 가존에 객체를 생성, 변경, 삭제한느 등의 기능을 수행하느 명령어를 데이터 정의어라고 한다

create table 소유계정, 테이블이름(
열1 이름 자료형,
열2 이름 자료형,
열3 이름 자료형,

열n 이름 자료형);

정수 타입 number(3)
문자타입 varchar2(5)

테이블 이름 생성 규칙
1. 테이블 이름은 무자로 사작해야한다 (한글도 가능하며 숫자로 시작할 수 없음)
2. 테이블 이름은 30byte 이하여야 한다 (즉 영어30자 , 한글은 15자 까지 사용가능)
3.같은 사용자 소유의 테이블 이름은 중복될 수 없다
4. 테이블 이름은 영문자 (한글 가능), 숫자(0~9)와 특수문자를 사용할 수 있다
5.sql 키원드는 열 이름으로 사용랗 수 없다(함수 이름 select,from 등은 열 이름으로 사용 불가)

열이름 생성 규칙
1. 열 이름은 문자로 사작해야 한다
2. 열 이름은 30byte 이하여야 한다
3. 한 테이블의 열 이름은 죽복될 수 없다
4. 열 이름은 영문자 (한글 가능), 숫자(0~9)와 특수문자를 사용할 수 있다
5.sql 키원드는 열 이름으로 사용랗 수 없다(함수 이름 select,from 등은 열 이름으로 사용 불가)
